# Simple-coder

Простая консольная утилита на C++ для шифрования и дешифрования текста, использующая подстановку через байтовый **S-Box** (аналогичный используемому в AES) и гаммирование (**XOR** операцию).

> ⚠️ ВНИМАНИЕ: Демонстрационная версия, не адаптирована под реальное использование

## Описание и Режимы работы

Приложение поддерживает два уровня криптостойкости (`behaviour`), которые выбираются при запуске:

* **Lite (Простой режим):**
  * Выполняет замену символов текста через таблицу подстановок `sBox`.
  * Применяет побитовое исключающее ИЛИ (XOR) с ключом, который циклически дублируется или обрезается под размер исходного текста.


* **Hard (Усиленный режим):**
  * Запускает цикл из **10 раундов** шифрования.
  * На каждом раунде ключ динамически модифицируется (сдвигается в зависимости от номера раунда), после чего последовательно применяются функции S-Box подстановки и XOR-гаммирования.



Для обоих режимов доступно как шифрование (`encrypt`), так и обратное дешифрование (`decrypt`) с использованием инвертированной таблицы `invsBox`.
